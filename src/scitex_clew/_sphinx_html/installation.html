Installation

Basic Installation

pip install scitex-clew

With CLI Support

pip install scitex-clew[cli]

With MCP Server

pip install scitex-clew[mcp]

All Optional Dependencies

pip install scitex-clew[all]

Development Installation

git clone https://github.com/ywatanabe1989/scitex-clew.git
cd scitex-clew
pip install -e ".[all]"
